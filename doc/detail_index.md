# 深度版学习结果索引

这组文档对应你前面已经确认过的 11 份“提纲版”学习文档，但目标完全不同。

- 提纲版解决的是“该从哪里看、该怎么学”。
- 深度版解决的是“我已经替你把代码读了一遍，现在把学习路线和题目答案直接展开给你”。

建议使用方式：

1. 先看原来的 `00_study_index.md` 和对应功能提纲，知道这部分在整机里的位置。
2. 再看这里的 `detail_XX_*.md`，把每一条学习路线直接转化成实现认知。
3. 如果你准备面试、汇报、做二次开发，就优先看每份文档里的“问题逐一回答”和“给小白的补充背景”。
